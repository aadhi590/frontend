## Project Summary
Bank Saathi is a premium fintech application inspired by CRED's design philosophy, integrated with an intelligent AI copilot. It focuses on trust-first design, progressive disclosure, and mobile-first interactions to help users manage their finances securely and intelligently.

## Tech Stack
- Framework: Next.js 15 (App Router)
- Styling: Tailwind CSS 4
- Animations: Framer Motion
- Icons: Lucide React
- Components: Radix UI primitives
- Charts: Recharts

## Architecture
- `src/app/`: Page routes and layouts
- `src/components/`: Shared UI components and feature-specific components
- `src/lib/`: Utility functions and hooks

## User Preferences
- Premium aesthetic (CRED-inspired)
- Deep Green and Gold color palette
- Neumorphic elements with soft shadows
- Mobile-first design

## Project Guidelines
- Use Tailwind CSS variables for theme consistency
- Implement animations for high-impact moments
- Maintain WCAG AA accessibility compliance
- No comments unless requested

## Common Patterns
- Progressive disclosure for technical details
- Graduated color coding for risk levels
- Timer-based interventions for security
